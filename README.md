# **🌿 JeeraBot**
### Chatbot for **SAS** 🛠️

---

### 📦 **Installation Instructions**

> First, install all the required packages with the following command:

```bash
pip install -r requirements.txt
```

---

### 🚀 **Run the Server**

> Use this command to start the server:

```bash
python main.py
```
